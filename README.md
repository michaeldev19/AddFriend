# AddFriend Addon

AddFriend is an enhanced World of Warcraft addon designed to streamline the process of managing your in-game social lists. It adds convenient "Add Friend," "Remove Friend," "Add to Guild," and "Remove from Guild" options directly to the right-click context menu of player unit frames.

## Features

This addon integrates essential social management options right where you need them:

- **Friend Management:** Adds "**Add Friend**" and "**Remove Friend**" option to the player unit context menu.
- **Guild Management (Requires appropriate Guild permissions):** Adds "**Invite to Guild**" and "**Remove from Guild**" option to the player unit context menu.

## Improvements

The following enhancements have been implemented in this version:

* **Interface Version Update:** The interface version has been updated to **30300**. This ensures full compatibility with all standard and "HD" client versions of **Wrath of the Lich King (3.3.5a)**.
* **Localization Support:** Added support for multiple languages, including **English** and **Spanish**.

## Usage

To start using AddFriend, follow these simple steps:

1.  Download the addon files.
2.  Place the `AddFriend` folder directly into your World of Warcraft `Interface/AddOns/` directory.
3.  Restart the WoW client or use the command `/reload` in-game to load the interface.
